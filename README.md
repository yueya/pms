# 权限管理系统

#### 项目介绍

  一个前后端分离的通用的权限管理系统，同时也是web项目的脚手架，可以很方便的基于此系统开发项目。各个web可独立启动，方便分模块开发。

 1 spring boot 不用介绍了  
 2 jooq orm框架  
 3 shiro 权限框架

 5 alibaba druib 数据库连接池
